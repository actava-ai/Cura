# Hermes Agent × Cura

Add Cura as a custom / OpenAI-compatible provider in the Hermes provider wizard:

```
# In the provider wizard (hermes model), add a custom / OpenAI-compatible provider:
Provider name:   cura
Base URL:        https://inference.actava.ai/v1
API key:         <your ACTAVA_API_KEY>
Default model:   actava/cura-soar

# Then start a session:
hermes
```
