# Cline (VS Code) × Cura

In Cline's settings, pick the **OpenAI Compatible** provider and fill in:

```
API Provider:            OpenAI Compatible
Base URL:                https://inference.actava.ai/v1
API Key:                 <your ACTAVA_API_KEY>
Model ID:                actava/cura-soar
Supports Images:         yes    (Cura 1T is text + vision)
Context Window:          256000
```
